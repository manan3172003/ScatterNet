# Welcome to ScatterNet by dodgerblue!

This is a distributed social networking app that lets you make blog posts, post photos, videos and
discover other authors doing the same!

This system also allows to connect to any other distributed Node also compliant with the API spec.

## Runbook

To get it running on your local system, setup the `.env` and the `.env.db` based on
the example files provided in the repository root.

Then simply invoke `docker compose up --build` and start signing up your users.
